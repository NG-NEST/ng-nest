## 添加文件结构

还是在 `api` 项目中，我们添加以下文件，通过文件夹来划分模块：

{{ __4\__api:src/system/users/user.module.ts:true:true }}

- `core` 公共的类型定义、服务等
- `system` 系统管理里面的模块
- `users` 指用户管理的模块
- `user.controller.ts` 控制器，处理请求并响应客户端
