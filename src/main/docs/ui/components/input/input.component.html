输入框 Input

通过键盘输入内容，是最基础的表单交互元素。

使用场景

当需要用户输入文字、字母、数字等内容时，就会用到输入框。

使用规范

  • 输入的内容不易过长，20个字符以下。
  • 如果内容是固定场景下的，需要做好各自的约定，比如只能输入字母或者数字。
