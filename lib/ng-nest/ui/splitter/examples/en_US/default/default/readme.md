Basic splitter usage.

- Use `x-splitter` to create a resizable split panel layout.
- Use `x-splitter-panel` to define each panel with size, min, and max constraints.
- Use `x-splitter-bar` to add a draggable divider between panels.
- Supports both horizontal (default) and vertical layouts via the `direction` property.
